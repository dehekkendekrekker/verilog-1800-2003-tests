====== cover property statement ======
Created Tuesday 16 December 2025

IEEE 1800-2023 Section A.2.10, 16.14

cover_property_statement ::=
	**cover property** ( [[property_spec]] ) [[6_Behavioral_statements/6.4_Statements:statement_or_null|statement_or_null]]

===== property_spec permutations =====

==== Permutation 1: property_expr only ====
cover property ( property_expr ) statement_or_null

==== Permutation 2: With clocking_event ====
cover property ( @(posedge clk) property_expr ) statement_or_null

==== Permutation 3: With disable iff ====
cover property ( disable iff (reset) property_expr ) statement_or_null

==== Permutation 4: With clocking_event and disable iff ====
cover property ( @(posedge clk) disable iff (reset) property_expr ) statement_or_null

===== statement_or_null permutations =====
* With pass statement
* Empty (null statement)

Note: Unlike assert/assume, cover only has pass action, no else clause.
